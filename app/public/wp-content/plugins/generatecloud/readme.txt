Host your own pattern cloud and share your patterns across your websites.

== Description ==

GenerateCloud allows you to set up your own cloud of patterns that you can easily share across your websites.

== Installation ==

To install GenerateCloud, simply upload the .zip file in "Plugins > Add New > Upload".
